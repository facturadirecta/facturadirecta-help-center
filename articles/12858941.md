<p class="no-margin">Si necesitas importar facturas de venta o de compra que incluyan <b>más de una línea de detalle</b>, actualmente la forma de hacerlo en FacturaDirecta es a través de nuestra <b>API</b>.</p><p class="no-margin"></p>
<p class="no-margin">La <a href="https://help.facturadirecta.com/es/articles/1070807-importar-facturas-y-gastos" target="_blank" class="intercom-content-link">importación mediante Excel/CSV</a> permite subir facturas sencillas, pero cuando la factura contiene líneas múltiples (por ejemplo, varios productos, servicios o conceptos), es necesario usar la API para generar la estructura completa de cada documento.</p><p class="no-margin"></p>
<p class="no-margin">En la documentación para desarrolladores encontrarás toda la información para crear facturas con varias líneas de forma automática:</p><p class="no-margin"></p>
