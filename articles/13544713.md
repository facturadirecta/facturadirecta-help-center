<p class="no-margin">El recurso <code>deliveryNotes</code> te permite gestionar los albaranes de entrega de tu empresa. Puedes crear albaranes, consultarlos, actualizarlos, eliminarlos y enviarlos por email.</p><p class="no-margin"></p>
<p class="no-margin">Los albaranes tienen una estructura muy similar a las facturas y presupuestos, por lo que si ya has trabajado con esos recursos, te resultará familiar.</p><p class="no-margin"></p>
<p class="no-margin">💡 <b>Recomendación</b>: Antes de empezar a desarrollar, te recomendamos <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">crear un entorno de pruebas (sandbox)</a> para hacer tus primeras llamadas sin afectar tus datos reales.</p><p class="no-margin"></p>

</h1><p class="no-margin">Los siguientes ejemplos muestran cómo crear albaranes para diferentes casos de uso. Todos los ejemplos usan el método POST a <code>/{companyId}/deliveryNotes</code>.</p><p class="no-margin"></p>
<h2 id="h_ejemplo_valorado">Albarán valorado</h2><p class="no-margin">Un albarán que incluye precios y servirá como base para generar una factura.</p><p class="no-margin"></p>
<pre><code>curl -X POST \  -H "facturadirecta-api-key: TU_CLAVE_API" \  -H "Content-Type: application/json" \  -d '{    "content": {      "type": "deliveryNote",      "main": {        "docNumber": {          "series": "AL"        },        "baseState": "pending",        "contact": "con_10000000-0000-4000-8000-000000000000",        "currency": "EUR",        "lines": [          {            "quantity": 1,            "unitPrice": 50,            "tax": ["S_IVA_21"],            "text": "Descripción del producto 1"          }        ]      }    }  }' \  https://app.facturadirecta.com/api/TU_COMPANY_ID/deliveryNotes</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Resultado:</b> Un albarán valorado con total 50€ + 21% IVA = 60,50€.</p><p class="no-margin"></p>
<h2 id="h_ejemplo_no_valorado">Albarán no valorado</h2><p class="no-margin">Un albarán que solo registra la entrega sin mostrar precios. Útil para entregas donde el precio se acordará después.</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "deliveryNote",    "main": {      "docNumber": {        "series": "AL"      },      "baseState": "pending",      "contact": "con_10000000-0000-4000-8000-000000000000",      "currency": "EUR",      "showPriceColumn": false,      "showAmountColumn": false,      "lines": [        {          "quantity": 5,          "unitPrice": 0,          "tax": ["S_IVA_21"],          "text": "Producto A - Referencia ABC123"        },        {          "quantity": 10,          "unitPrice": 0,          "tax": ["S_IVA_21"],          "text": "Producto B - Referencia XYZ789"        }      ]    }  }}</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Opciones de visualización:</b></p><ul><li><p class="no-margin"><code>showPriceColumn: false</code> oculta la columna de precio unitario</p></li><li><p class="no-margin"><code>showAmountColumn: false</code> oculta la columna de importe</p></li></ul><h2 id="h_ejemplo_multilinea">Albarán con múltiples productos</h2><p class="no-margin">Un albarán típico de entrega de mercancía.</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "deliveryNote",    "main": {      "docNumber": {        "series": "AL"      },      "baseState": "delivered",      "contact": "con_10000000-0000-4000-8000-000000000000",      "date": "2026-01-28",      "currency": "EUR",      "lines": [        {          "quantity": 100,          "unitPrice": 2.50,          "tax": ["S_IVA_21"],          "text": "Producto A - Caja de 100 unidades"        },        {          "quantity": 50,          "unitPrice": 4.00,          "tax": ["S_IVA_21"],          "text": "Producto B - Pack de 50 unidades"        },        {          "quantity": 25,          "unitPrice": 8.00,          "tax": ["S_IVA_10"],          "text": "Producto C - IVA reducido"        }      ]    }  }}</code></pre><p class="no-margin"></p>

<h1 id="h_convertir">Convertir albarán a factura</h1><p class="no-margin">Cuando quieras facturar los productos entregados, puedes crear una factura referenciando el albarán. El flujo típico es:</p><ol><li><p class="no-margin">Actualizar el albarán a estado "delivered"</p></li><li><p class="no-margin">Crear una factura con las líneas del albarán </p></li></ol><p class="no-margin">Al crear la factura, indica el origen de cada línea para vincular los documentos:</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "invoice",    "main": {      "contact": "con_xxxxx",      "currency": "EUR",      "lines": [        {          "origin": "del_xxxxx",          "quantity": 100,          "unitPrice": 2.50,          "tax": ["S_IVA_21"],          "text": "Producto A - Albarán AL-2026-001"        }      ]    }  }}</code></pre><p class="no-margin"></p>
<h1 id="h_errores">Errores comunes </h1><h2 id="h_error_contact">Error: "contact is required" </h2><p class="no-margin">El campo <code>contact</code> es obligatorio. Asegúrate de incluir el ID de un cliente existente.</p><p class="no-margin"></p>
<h2 id="h_error_state">Error: "Invalid baseState"</h2><p class="no-margin">El valor de <code>baseState</code> no es válido. Usa uno de: pending, delivered, closed, rejected.</p><p class="no-margin"></p>
<h2 id="h_error_lines">Error: "lines must have at least 1 item"</h2><p class="no-margin">Un albarán necesita al menos una línea. Verifica que el array <code>lines</code> no está vacío.</p><p class="no-margin"></p>
