<p class="no-margin">Las claves de API te permiten conectar aplicaciones externas con tu cuenta de FacturaDirecta de forma segura. Cada clave tiene permisos específicos y da acceso únicamente a la empresa donde se creó.</p><p class="no-margin"></p>
<h1 id="h_acceder_configuracion">Acceder a la configuración de API</h1><p class="no-margin">Para gestionar tus claves de API:</p><ol><li><p class="no-margin">Haz clic en ⚙️ <b>Ajustes</b> (icono de engranaje en el menú superior)</p></li><li><p class="no-margin">Selecciona <b>"API e integraciones"</b> en el menú lateral</p></li></ol><p class="no-margin"></p>
<h1 id="h_obtener_company_id">Obtener tu identificador de empresa (companyId)</h1><p class="no-margin">En la sección de API verás tu <b>identificador de empresa</b> (companyId). Este valor es único para cada empresa y lo necesitarás en todas las llamadas a la API.</p><p class="no-margin"></p>
<p class="no-margin">El companyId aparece en la URL de la API:</p><pre><code>https://app.facturadirecta.com/api/{companyId}/invoices</code></pre><p class="no-margin"></p>
<h1 id="h_crear_clave">Crear una nueva clave de API</h1><ol><li><p class="no-margin">Haz clic en <b>"Crear nueva clave de API"</b></p></li><li><p class="no-margin">Introduce un <b>nombre</b> descriptivo para identificar el uso de la clave (por ejemplo: "Integración CRM" o "App móvil")</p></li><li><p class="no-margin">Selecciona los <b>permisos</b> que necesitas:</p><ul><li><p class="no-margin"><b>TODOS</b>: Acceso completo de lectura y escritura</p></li><li><p class="no-margin"><b>LECTURA</b>: Solo lectura en todos los recursos</p></li><li><p class="no-margin"><b>Permisos individuales</b>: Selecciona los permisos específicos que necesitas</p></li></ul></li><li><p class="no-margin">Haz clic en <b>"Crear clave"</b></p></li><li><p class="no-margin"><b>Copia y guarda la clave inmediatamente</b>: por seguridad, solo se muestra una vez</p></li></ol><p class="no-margin">⚠️ <b>Importante</b>: Guarda tu clave de API de forma segura. No podrás volver a verla después de cerrar el diálogo. Si la pierdes, tendrás que crear una nueva.</p><p class="no-margin"></p>

<h1 id="h_eliminar_clave">Eliminar una clave</h1><ol><li><p class="no-margin">Haz clic en la clave que quieres eliminar</p></li><li><p class="no-margin">En el menú de acciones, selecciona <b>"Borrar"</b></p></li><li><p class="no-margin">Confirma la eliminación</p></li></ol><p class="no-margin">⚠️ <b>Atención</b>: Tras eliminar una clave, cualquier aplicación que la use dejará de funcionar inmediatamente. Asegúrate de actualizar primero tus integraciones con una nueva clave.</p><p class="no-margin"></p>
<h1 id="h_notas_seguridad">Notas de seguridad</h1><ul><li><p class="no-margin"><b>No compartas tus claves</b>: Trátalas como contraseñas</p></li><li><p class="no-margin"><b>Usa permisos mínimos</b>: Asigna solo los permisos que la integración necesita</p></li><li><p class="no-margin"><b>Rota las claves periódicamente</b>: Si sospechas que una clave está comprometida, elimínala y crea una nueva</p></li><li><p class="no-margin"><b>Una clave por integración</b>: Facilita identificar y revocar accesos si es necesario</p></li></ul><p class="no-margin">💡 <b>Tip</b>: Las claves API del <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">sandbox</a> son independientes de las de producción. Crea claves distintas para cada entorno y no las mezcles.</p>
