<p class="no-margin">La verificación en dos pasos mediante una aplicación de autenticación permite aumentar la seguridad de tu cuenta, puesto que para acceder al programa será necesario indicar, además de tu usuario y contraseña, un código que se generará en tu dispositivo móvil y que cambia cada 30 segundos; de esta forma puedes evitar que alguien no autorizado acceda a tu cuenta incluso si tu contraseña se ha visto comprometida de alguna forma.</p><p class="no-margin">Puedes utilizar como aplicaciones para generar estos códigos las aplicaciones siguientes cuya compatibilidad ha sido probada con FacturaDirecta:</p><ul><li><p class="no-margin">Free OTP para <a href="https://play.google.com/store/apps/details?id=org.fedorahosted.freeotp&amp;hl=es" target="_blank" class="intercom-content-link">Android</a> o para <a href="https://apps.apple.com/es/app/freeotp-authenticator/id872559395" target="_blank" class="intercom-content-link">iOS</a></p></li><li><p class="no-margin">Google Authenticator para <a href="https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2&amp;hl=es" target="_blank" class="intercom-content-link">Android</a> o para <a href="https://apps.apple.com/es/app/google-authenticator/id388497605" target="_blank" class="intercom-content-link">iOS</a></p></li></ul><p class="no-margin">También es posible utilizar cualquier otra aplicación estándar que permita la generación de claves de verificación de tipo TOTP; durante el proceso de configuración podrás comprobar si es compatible con FacturaDirecta.</p><h1 id="h_430358efcc">¿Cómo se activa esta verificación?</h1><p class="no-margin">1. Accede a la página de gestión del perfil de tu usuario desde el icono con las siglas en la parte superior derecha y dale a "Perfil de usuairo"</p><p class="no-margin"></p>
<div class="intercom-container intercom-align-center"><img src="https://downloads.intercomcdn.com/i/o/ind32iz6/1697237922/058521768200249de71d19c8e5f6/siglas+-+Perfil+de+usuario.png" alt="Perfil de usuario"></div><p class="no-margin"></p>
<p class="no-margin">2. Dale al<b> "Autenticador"</b> en el menú del lateral izquierdo:</p><p class="no-margin"></p>
<div class="intercom-container intercom-align-center"><img src="https://downloads.intercomcdn.com/i/o/1028615785/87e4e8df90f1e8efa2083d33/Acceso+autenticador.png" alt="Acceso a la configuración del autenticador"></div><p class="no-margin"></p>
<p class="no-margin">3. Irás a la página de configuración, donde te explica los pasos que tienes que seguir.</p><p class="no-margin"></p>
<div class="intercom-container intercom-align-center"><img src="https://downloads.intercomcdn.com/i/o/1028799108/9253be8b8093f6122137792c/Configuracio%CC%81n+del+autenticador.png" alt="Configuración del dispositivo autenticador"></div><p class="no-margin"></p>
<p class="no-margin">Solo tienes que escanear con la aplicación de autenticación el código QR que ves en la página, si la aplicación que has elegido no permite escanear un código también puedes teclear dar de alta el acceso tecleando manualmente el código visible por debajo del código.</p><p class="no-margin">A continuación tienes teclear el código que se está generando en ese momento y asignar un nombre al dispositivo que estás usando (por si quieres utilizar más de uno).</p><p class="no-margin"><b>IMPORTANTE: </b>Si te equivocas al introducir el código se generará un nuevo código QR distinto y tendrás que volver escanearlo.</p><p class="no-margin">Una vez finalizada la configuración después de introducir tu usuario y contraseña correctos tendrás que introducir el código que se genere en ese momento en tu dispositivo en una pantalla como esta.</p><p class="no-margin"></p>
<div class="intercom-container intercom-align-center"><img src="https://downloads.intercomcdn.com/i/o/1028840019/c976d8fb3b05e8c7eb9cd013/Introduccio%CC%81n+del+co%CC%81digo+OTP.png" alt="Introducción del código OTP"></div><p class="no-margin"></p>
<p class="no-margin"><b>IMPORTANTE: </b>Fíjate bien que el mensaje te indique que debes introducir el código generado en tu dispositivo y no uno enviado por correo electrónico.</p><p class="no-margin"></p>
