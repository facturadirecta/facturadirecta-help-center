<p class="no-margin">Cuando un contacto tiene el rol de <b>empleado</b>, puedes configurar sus preferencias para automatizar el registro de nóminas.</p><p class="no-margin"></p>
<h1 id="h_como_configurar_las_preferencias">Cómo configurar las preferencias</h1><p class="no-margin">1. Abre la ficha del contacto.</p><p class="no-margin">2. Busca la tarjeta <b>"Preferencias del empleado"</b>.</p><p class="no-margin">3. Haz clic en el icono de edición de esa tarjeta.</p><p class="no-margin">4. Configura las opciones según tus necesidades.</p><p class="no-margin">5. Haz clic en <b>"Guardar y cerrar"</b>.</p><p class="no-margin"></p>
<h1 id="h_campos_disponibles">Campos disponibles </h1><h2 id="h_bc7fae42e1">Método de pago preferido</h2><p class="no-margin">Configura cómo pagas habitualmente a este empleado:</p><ul><li><p class="no-margin"><b>Transferencia</b>: Realizas una transferencia bancaria al empleado.</p></li><li><p class="no-margin"><b>Transferencia SEPA</b>: Transferencia bancaria en formato SEPA, permite generar ficheros de remesas para enviar al banco.</p></li><li><p class="no-margin"><b>Tarjeta</b>: Pago con tarjeta de crédito o débito.</p></li><li><p class="no-margin"><b>Efectivo</b>: Pago en metálico.</p></li><li><p class="no-margin"><b>Otros</b>: Cheque, pagaré, etc.</p></li></ul><p class="no-margin">Al configurar este campo, las nóminas que registres de este empleado incluirán automáticamente la forma de pago correcta.</p><p class="no-margin"></p>
<h2 id="h_cuenta_contable">Cuenta contable (si se ha contratado contabilidad)</h2><p class="no-margin">Asigna la cuenta contable de empleado para la contabilidad:</p><ul><li><p class="no-margin">Por defecto se usa la cuenta 465 (Remuneraciones pendientes de pago).</p></li><li><p class="no-margin">Puedes crear subcuentas específicas por empleado para un mayor detalle contable.</p></li><li><p class="no-margin">Esta cuenta se usará en los asientos contables de las nóminas.</p></li></ul><h1 id="h_ventajas_de_configurar_preferencias">Ventajas de configurar preferencias</h1><ul><li><p class="no-margin"><b>Ahorra tiempo:</b> Los datos se rellenan automáticamente al registrar nóminas.</p></li><li><p class="no-margin"><b>Evita errores:</b> Siempre se aplican las mismas configuraciones.</p></li><li><p class="no-margin"><b>Mejora la contabilidad:</b> Las cuentas contables se asignan correctamente.</p></li></ul><p class="no-margin"></p>
<h1 id="h_nota_importante">Nota importante</h1><p class="no-margin">⚠️ Las preferencias configuradas se aplican a las <b>nuevas nóminas</b> que registres a partir de ese momento. Las nóminas ya registradas no se modifican automáticamente.</p><p class="no-margin"></p>
<h1 id="h_diferencia_con_otros_roles">Diferencia con otros roles</h1><p class="no-margin">Un empleado no puede tener más roles asociados (no puede ser ni cliente ni proveedor).</p>
