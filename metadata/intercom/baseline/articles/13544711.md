<p class="no-margin">El recurso <code>bills</code> te permite consultar las facturas de compra y gastos (tickets) de tu empresa. Puedes listar todas las facturas, filtrar por diferentes criterios y obtener los detalles de una factura específica.</p><p class="no-margin"></p>
<p class="no-margin">Este artículo incluye ejemplos prácticos para consultar facturas de compra usando los diferentes filtros disponibles.</p><p class="no-margin"></p>
<p class="no-margin"><b>Nota:</b> Actualmente la API permite consultar facturas de compra pero no crearlas. Para registrar gastos, usa la interfaz web de FacturaDirecta o el escáner de documentos.</p><p class="no-margin"></p>
<p class="no-margin">💡 <b>Recomendación</b>: Antes de empezar a desarrollar, te recomendamos <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">crear un entorno de pruebas (sandbox)</a> para hacer tus primeras llamadas sin afectar tus datos reales.</p><p class="no-margin"></p>
<h1 id="h_endpoints">Endpoints disponibles</h1><div class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Método</b></p></td><td><p class="no-margin"><b>Endpoint</b></p></td><td><p class="no-margin"><b>Descripción</b></p></td></tr><tr><td><p class="no-margin">GET</p></td><td><p class="no-margin">/{companyId}/bills</p></td><td><p class="no-margin">Lista de facturas de compra (con filtros y paginación)</p></td></tr><tr><td><p class="no-margin">GET</p></td><td><p class="no-margin">/{companyId}/bills/{id}</p></td><td><p class="no-margin">Obtener una factura de compra por ID</p></td></tr></tbody></table></div><h1 id="h_ejemplos">Ejemplos de código</h1><h2 id="h_ejemplo_list">Listar todas las facturas de compra</h2><p class="no-margin">Obtiene las facturas de compra más recientes con paginación por defecto.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  https://app.facturadirecta.com/api/TU_COMPANY_ID/bills</code></pre><p class="no-margin"></p>
<h2 id="h_ejemplo_dates">Filtrar por rango de fechas </h2><p class="no-margin">Obtiene las facturas de compra de un periodo específico.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  "https://app.facturadirecta.com/api/TU_COMPANY_ID/bills?minDate=2026-01-01&amp;maxDate=2026-01-31"</code></pre><p class="no-margin"></p>
<h2 id="h_ejemplo_contact">Filtrar por proveedor </h2><p class="no-margin">Obtiene las facturas de compra de un proveedor específico.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  "https://app.facturadirecta.com/api/TU_COMPANY_ID/bills?contact=con_xxxxx"</code></pre><p class="no-margin"></p>
<h2 id="h_ejemplo_amount">Filtrar por importe</h2><p class="no-margin">Obtiene las facturas de compra con importe entre 100€ y 500€.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  "https://app.facturadirecta.com/api/TU_COMPANY_ID/bills?minTotal=100&amp;maxTotal=500"</code></pre><p class="no-margin"></p>
<h2 id="h_ejemplo_nocontact">Buscar tickets sin proveedor</h2><p class="no-margin">Útil para encontrar gastos menores (tickets) que no tienen proveedor asociado.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  "https://app.facturadirecta.com/api/TU_COMPANY_ID/bills?hasContact=false"</code></pre><p class="no-margin"></p>
<h2 id="h_ejemplo_tags">Filtrar por etiquetas</h2><p class="no-margin">Obtiene las facturas que tienen una etiqueta específica.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  "https://app.facturadirecta.com/api/TU_COMPANY_ID/bills?anyOfTheseTags=tag_xxxxx"</code></pre><p class="no-margin"></p>
<h2 id="h_ejemplo_detail">Obtener detalle de una factura</h2><p class="no-margin">Obtiene todos los datos de una factura de compra específica.</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \  https://app.facturadirecta.com/api/TU_COMPANY_ID/bills/bil_xxxxx</code></pre><p class="no-margin"></p>
