<p class="no-margin">El recurso <code>estimates</code> te permite gestionar los presupuestos de tu empresa. Puedes crear presupuestos, consultarlos, actualizarlos, eliminarlos y enviarlos por email.</p><p class="no-margin"></p>
<p class="no-margin">Los presupuestos tienen una estructura muy similar a las facturas de venta, por lo que si ya has trabajado con el recurso <code>invoices</code>, te resultará familiar.</p><p class="no-margin"></p>
<p class="no-margin">💡 <b>Recomendación</b>: Antes de empezar a desarrollar, te recomendamos <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">crear un entorno de pruebas (sandbox)</a> para hacer tus primeras llamadas sin afectar tus datos reales.</p><p class="no-margin"></p>
<h1 id="h_endpoints">Endpoints disponibles</h1><div class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Método</b></p></td><td><p class="no-margin"><b>Endpoint</b></p></td><td><p class="no-margin"><b>Descripción</b></p></td></tr><tr><td><p class="no-margin">GET</p></td><td><p class="no-margin">/{companyId}/estimates</p></td><td><p class="no-margin">Lista de presupuestos (con filtros y paginación)</p></td></tr><tr><td><p class="no-margin">POST</p></td><td><p class="no-margin">/{companyId}/estimates</p></td><td><p class="no-margin">Crear un presupuesto</p></td></tr><tr><td><p class="no-margin">GET</p></td><td><p class="no-margin">/{companyId}/estimates/{id}</p></td><td><p class="no-margin">Obtener un presupuesto por ID</p></td></tr><tr><td><p class="no-margin">PUT</p></td><td><p class="no-margin">/{companyId}/estimates/{id}</p></td><td><p class="no-margin">Actualizar un presupuesto</p></td></tr><tr><td><p class="no-margin">DELETE</p></td><td><p class="no-margin">/{companyId}/estimates/{id}</p></td><td><p class="no-margin">Eliminar un presupuesto</p></td></tr><tr><td><p class="no-margin">PUT</p></td><td><p class="no-margin">/{companyId}/estimates/{id}/pdf</p></td><td><p class="no-margin">Generar PDF del presupuesto</p></td></tr><tr><td><p class="no-margin">PUT</p></td><td><p class="no-margin">/{companyId}/estimates/{id}/send</p></td><td><p class="no-margin">Enviar presupuesto por email</p></td></tr><tr><td><p class="no-margin">PUT</p></td><td><p class="no-margin">/{companyId}/estimates/{id}/tags</p></td><td><p class="no-margin">Actualizar etiquetas del presupuesto</p></td></tr></tbody></table></div><h1 id="h_ejemplos">Ejemplos de código</h1><p class="no-margin">Los siguientes ejemplos muestran cómo crear presupuestos para diferentes casos de uso. Todos los ejemplos usan el método POST a <code>/{companyId}/estimates</code>.</p><p class="no-margin"></p>
<h2 id="h_ejemplo_minimal">Presupuesto básico </h2><p class="no-margin">El caso más simple: un presupuesto con una línea y formato estándar.</p><p class="no-margin"></p>
<pre><code>curl -X POST \  -H "facturadirecta-api-key: TU_CLAVE_API" \  -H "Content-Type: application/json" \  -d '{    "content": {      "type": "estimate",      "main": {        "docNumber": {          "series": "P"        },        "baseState": "pending",        "contact": "con_10000000-0000-4000-8000-000000000000",        "currency": "EUR",        "lines": [          {            "quantity": 1,            "unitPrice": 100,            "tax": ["S_IVA_21"],            "text": "Descripción del servicio"          }        ]      }    }  }' \  https://app.facturadirecta.com/api/TU_COMPANY_ID/estimates</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Resultado:</b> Un presupuesto de 100€ + 21% IVA = 121€ total, en estado "Pendiente".</p><p class="no-margin"></p>
<h2 id="h_ejemplo_description">Presupuesto con formato simplificado </h2><p class="no-margin">Útil cuando solo quieres mostrar descripciones sin columnas de cantidad o producto.</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "estimate",    "main": {      "showProductColumn": false,      "showQuantityColumn": false,      "docNumber": {        "series": "P"      },      "baseState": "pending",      "contact": "con_10000000-0000-4000-8000-000000000000",      "currency": "EUR",      "lines": [        {          "quantity": 1,          "unitPrice": 100,          "lineTotal": 100,          "tax": ["S_IVA_21"],          "text": "Descripción del servicio"        }      ]    }  }}</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Opciones de visualización:</b></p><ul><li><p class="no-margin"><code>showProductColumn: false</code> oculta la columna de producto/referencia</p></li><li><p class="no-margin"><code>showQuantityColumn: false</code> oculta la columna de cantidad</p></li></ul><h2 id="h_ejemplo_discount">Presupuesto con descuento</h2><p class="no-margin">Aplica un descuento a una línea del presupuesto.</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "estimate",    "main": {      "docNumber": {        "series": "P"      },      "baseState": "pending",      "contact": "con_10000000-0000-4000-8000-000000000000",      "currency": "EUR",      "lines": [        {          "quantity": 1,          "unitPrice": 100,          "discountRate": 10,          "tax": ["S_IVA_21"],          "text": "Servicio de consultoría (10% descuento)"        }      ]    }  }}</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Resultado:</b></p><ul><li><p class="no-margin">Precio: 100€</p></li><li><p class="no-margin">Descuento 10%: -10€</p></li><li><p class="no-margin">Base: 90€</p></li><li><p class="no-margin">IVA 21%: 18,90€</p></li><li><p class="no-margin"><b>Total:</b> 108,90€</p></li></ul><h2 id="h_ejemplo_validity">Presupuesto con validez</h2><p class="no-margin">Indica la fecha de validez del presupuesto.</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "estimate",    "main": {      "docNumber": {        "series": "P"      },      "baseState": "pending",      "contact": "con_10000000-0000-4000-8000-000000000000",      "date": "2026-01-28",      "dueDate": "2026-02-28",      "currency": "EUR",      "lines": [        {          "quantity": 1,          "unitPrice": 500,          "tax": ["S_IVA_21"],          "text": "Desarrollo de aplicación web"        }      ]    }  }}</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Nota:</b> El campo <code>dueDate</code> en presupuestos indica la fecha de validez, no la fecha de vencimiento de pago.</p><p class="no-margin"></p>

class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Estado</b></p></td><td><p class="no-margin"><b>Descripción</b></p></td></tr><tr><td><p <h1 id="h_convertir">Convertir presupuesto a factura</h1><p class="no-margin">Cuando un cliente acepta un presupuesto, puedes convertirlo a factura. El flujo típico es:</p><ol><li><p class="no-margin">Actualizar el presupuesto a estado "accepted"</p></li><li><p class="no-margin">Crear una factura con las mismas líneas y referenciar el presupuesto</p></li></ol><p class="no-margin">Al crear la factura, puedes indicar el origen de cada línea para que queden vinculados:</p><p class="no-margin"></p>
<pre><code>{  "content": {    "type": "invoice",    "main": {      "contact": "con_xxxxx",      "currency": "EUR",      "lines": [        {          "origin": "est_xxxxx",          "quantity": 1,          "unitPrice": 100,          "tax": ["S_IVA_21"],          "text": "Servicio aprobado en presupuesto"        }      ]    }  }}</code></pre><p class="no-margin"></p>
<h1 id="h_errores">Errores comunes</h1><h2 id="h_error_contact">Error: "contact is required" </h2><p class="no-margin">El campo <code>contact</code> es obligatorio. Asegúrate de incluir el ID de un cliente existente.</p><p class="no-margin"></p>
<h2 id="h_error_state">Error: "Invalid baseState"</h2><p class="no-margin">El valor de <code>baseState</code> no es válido. Usa uno de: pending, accepted, rejected, closed.</p><p class="no-margin"></p>
<h2 id="h_error_lines">Error: "lines must have at least 1 item" </h2><p class="no-margin">Un presupuesto necesita al menos una línea. Verifica que el array <code>lines</code> no está vacío.</p><p class="no-margin"></p>
