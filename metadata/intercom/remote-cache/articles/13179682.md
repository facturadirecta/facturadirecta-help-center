<p class="no-margin">En <b>FacturaDirecta</b>, utilizamos identificadores internos para organizar cada elemento de tu cuenta. Si intentas acceder a un enlace y el programa te indica que el registro no existe, te explicamos por qué puede estar ocurriendo.</p><p class="no-margin"></p>
<h2 id="h_241f2de7e2">¿Qué son estos identificadores?</h2><p class="no-margin">Los códigos que ves en la URL de tu navegador son etiquetas internas que el sistema asigna a cada apartado. Algunos de los más comunes son:</p><ul><li><p class="no-margin"><b>inv_</b>: Identificador para facturas de venta.</p></li><li><p class="no-margin"><b>bill_</b>: Identificador para facturas de compra.</p></li><li><p class="no-margin"><b>est_</b>: Identificador para presupuestos.</p></li><li><p class="no-margin"><b>con_</b>: Identificador para contactos.</p></li></ul><h2 id="h_eef83982ff">¿Por qué aparece el error de que "no existe"?</h2><p class="no-margin">Existen dos motivos principales por los que el sistema no encuentra un registro específico:</p><ol><li><p class="no-margin"><b>El documento no se llegó a crear:</b> Es la causa más frecuente. Si tenías abierta la pantalla de creación de una factura o presupuesto, el sistema genera un ID temporal en la URL. Si cerraste la ventana o navegaste hacia atrás sin pulsar en "Guardar" o "Crear", ese documento nunca llegó a existir en nuestra base de datos. Al intentar reutilizar ese enlace más tarde, el programa te informa de que el registro no existe porque no se completó su creación.</p></li><li><p class="no-margin"><b>Falta de permisos o acceso:</b> Otra opción es que el documento sí exista, pero tu usuario actual no tenga los permisos necesarios para visualizarlo o pertenezca a una cuenta de empresa diferente.</p></li></ol><p class="no-margin">Si crees que el documento debería estar ahí, te recomendamos buscarlo directamente en el listado correspondiente desde el menú principal para confirmar si se guardó correctamente.</p>
