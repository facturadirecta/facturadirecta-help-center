<p class="no-margin"></p>
<p class="no-margin">Las etiquetas son una herramienta muy útil para organizar y clasificar tus documentos en FacturaDirecta. Te permiten crear tu propio sistema de categorización adaptado a las necesidades de tu negocio.</p><p class="no-margin"></p>
<h1 id="h_que_son_etiquetas">¿Qué son las etiquetas?</h1><p class="no-margin">Las etiquetas son palabras o frases cortas que puedes asignar a tus documentos para clasificarlos. Por ejemplo, puedes usar etiquetas como:</p><ul><li><p class="no-margin"><b>Por proyecto</b>: "Proyecto Web", "Reforma Local", "Cliente Nuevo"</p></li><li><p class="no-margin"><b>Por estado</b>: "Urgente", "Pendiente revisión", "Para enviar"</p></li><li><p class="no-margin"><b>Por categoría</b>: "Mantenimiento", "Consultoría", "Formación"</p></li><li><p class="no-margin"><b>Por origen</b>: "Tienda online", "Feria 2024", "Recomendación"</p></li><li><p class="no-margin"></p>
</li></ul><div class="intercom-container"><img src="https://downloads.intercomcdn.com/i/o/ind32iz6/1892746858/2d66d9d828a7b6ab40f2c7d21cd8/etiquetas-ejemplo.jpg" width="485" height="88" style="height: auto;"></div><p class="no-margin"></p>
<p class="no-margin"></p>
<h1 id="h_para_que_sirven">¿Para qué sirven las etiquetas?</h1><p class="no-margin">Las etiquetas te ayudan a:</p><ul><li><p class="no-margin"><b>Organizar</b> tus documentos según tus propios criterios</p></li><li><p class="no-margin"><b>Filtrar</b> listados para ver solo los documentos con determinadas etiquetas</p></li><li><p class="no-margin"><b>Generar informes</b> filtrados por etiquetas específicas</p></li><li><p class="no-margin"><b>Encontrar documentos</b> relacionados rápidamente</p></li></ul><h1 id="h_documentos_compatibles">¿En qué documentos puedo usar etiquetas?</h1><p class="no-margin">Las etiquetas están disponibles en los siguientes tipos de documentos:</p><ul><li><p class="no-margin">Facturas de venta</p></li><li><p class="no-margin">Presupuestos</p></li><li><p class="no-margin">Albaranes</p></li><li><p class="no-margin">Gastos y facturas de compra</p></li><li><p class="no-margin">Pagos</p></li><li><p class="no-margin">Nóminas</p></li><li><p class="no-margin">Impuestos</p></li><li><p class="no-margin">Contactos (clientes, proveedores, etc.)</p></li><li><p class="no-margin">Transacciones bancarias</p></li><li><p class="no-margin">Tareas recurrentes</p></li></ul><h1 id="h_crear_etiquetas">¿Cómo se crean las etiquetas?</h1><p class="no-margin">Las etiquetas se crean <b>automáticamente</b> cuando las asignas a un documento por primera vez. No necesitas ir a ninguna sección de configuración para crear etiquetas nuevas.</p><p class="no-margin"></p>
<p class="no-margin">Simplemente, cuando vayas a añadir una etiqueta a un documento, escribe el nombre de la etiqueta que quieres crear y pulsa Enter. La etiqueta se creará y quedará disponible para usar en otros documentos.</p><p class="no-margin"></p>
<p class="no-margin">Una vez que una etiqueta ya no está asignada a ningún documento, desaparece automáticamente del sistema.</p><p class="no-margin"></p>
<p class="no-margin"><b>Nota importante:</b> Los colores de las etiquetas se generan automáticamente por el sistema y <b>no pueden ser modificados manualmente</b>.</p><p class="no-margin"></p>
