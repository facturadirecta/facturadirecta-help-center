<p class="no-margin">En el nuevo <b>FacturaDirecta</b>, la gestión de tus contactos ha evolucionado para ser más ágil. Si buscas el antiguo número de cliente del programa clásico, aquí te detallamos las opciones disponibles actualmente:</p><p class="no-margin"></p>
<h2 id="h_19499b81e9">Código para exportación (Contabilidad externa)</h2><p class="no-margin">En <b>FacturaDirecta</b>, dispones del <b>"Código cliente para exportación"</b>. Es importante que sepas que este código se utiliza únicamente cuando exportas tu contabilidad a programas externos como A3 o Contaplus.</p><p class="no-margin"></p>
<div class="intercom-container"><img src="https://downloads.intercomcdn.com/i/o/ind32iz6/1890712658/a02e9d9861db5ee084d5a8e54683/codigo-cliente-para-exportacion.jpg" width="923" height="405" style="height: auto;"></div><p class="no-margin"></p>
<ul><li><p class="no-margin">Este código no se usa internamente para procesos dentro de <b>FacturaDirecta</b>.</p><p class="no-margin"></p>
</li><li><p class="no-margin">Al realizar una exportación, el sistema genera este código de forma secuencial y automática.</p><p class="no-margin"></p>
</li><li><p class="no-margin">Dispones de una opción simétrica para tus proveedores denominada <b>"Código proveedor para exportación"</b>.</p></li></ul><h2 id="h_a3f54a700e">Identificador externo</h2><p class="no-margin">Si necesitas un campo para asignar un número manual o una referencia propia a tus clientes, puedes utilizar el <b>"Identificador externo"</b>.</p><ul><li><p class="no-margin">Encontrarás este campo libre dentro de la ficha de cada contacto.</p><p class="no-margin"></p>
</li><li><p class="no-margin">Está ubicado justo debajo del apartado de <b>"Notas internas"</b>.</p><p class="no-margin"></p>
</li><li><p class="no-margin">Puedes usarlo con total libertad para añadir la numeración que prefieras y mantener el orden de tu negocio.</p></li></ul><div class="intercom-container"><img src="https://downloads.intercomcdn.com/i/o/ind32iz6/1890713083/ade336104d34a41c1640975083f0/identificador-externo.jpg" width="921" height="481" style="height: auto;"></div><p class="no-margin"></p>
