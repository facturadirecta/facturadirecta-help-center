<p class="no-margin">La API pública de FacturaDirecta te permite integrar tu software con el sistema de facturación. Automatiza la gestión de tu negocio conectando FacturaDirecta con otras herramientas: crea facturas automáticamente desde tu CRM, sincroniza contactos con tu ERP, o genera informes personalizados con los datos de tu contabilidad.</p><p class="no-margin"></p>
<p class="no-margin">💡 <b>Recomendación</b>: Antes de empezar a desarrollar, te recomendamos <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">crear un entorno de pruebas (sandbox)</a> para hacer tus primeras llamadas sin afectar tus datos reales.</p><p class="no-margin"></p>

<h1 id="h_autenticacion">Autenticación</h1><p class="no-margin">Hay dos formas de autenticarte en la API:</p><p class="no-margin"></p>
<h2 id="h_api_key">1. API Key (recomendado para integraciones)</h2><p class="no-margin">Crea una clave de API desde FacturaDirecta y añádela en la cabecera de tus peticiones:</p><p class="no-margin"></p>
<pre><code>facturadirecta-api-key: TU_CLAVE_API</code></pre><p class="no-margin"></p>
<p class="no-margin"><b>Ventajas</b>:</p><ul><li><p class="no-margin">El acceso se controla por los permisos asignados a la clave</p></li><li><p class="no-margin">Cada clave da acceso únicamente a la empresa donde se creó</p></li><li><p class="no-margin">No depende de ningún usuario (no se ve afectada si se elimina un usuario)</p></li></ul><p class="no-margin">👉 <a href="https://help.facturadirecta.com/es/articles/13538824-como-crear-claves-de-api" target="_blank" class="intercom-content-link">Cómo crear claves de API</a></p><p class="no-margin"></p>
<h2 id="h_oauth2">2. OAuth2</h2><p class="no-margin">Utiliza el flujo de Authorization Code para aplicaciones que necesitan acceder a múltiples empresas.</p><p class="no-margin"></p>
<p class="no-margin"><b>Ventajas</b>:</p><ul><li><p class="no-margin">Permite acceder a múltiples empresas con las mismas credenciales</p></li><li><p class="no-margin">El acceso combina los permisos del usuario y los scopes solicitados</p></li></ul><p class="no-margin">👉 <a href="https://help.facturadirecta.com/es/articles/13538825-autenticacion-oauth2-para-la-api" target="_blank" class="intercom-content-link">Autenticación OAuth2 para la API</a></p><p class="no-margin"></p>
<h1 id="h_webhooks">Webhooks</h1><p class="no-margin">Los webhooks te permiten recibir notificaciones en tiempo real cada vez que ocurre algo relevante en tu cuenta: una factura creada, un contacto modificado, un gasto archivado y más. En lugar de consultar la API repetidamente, FacturaDirecta envía la información directamente a tu servidor en el momento en que se produce el evento.</p><p class="no-margin"></p>
<p class="no-margin">⚠️ Los webhooks están disponibles en los planes <b>Avanzado</b> y <b>Total</b>.</p><p class="no-margin"></p>
<p class="no-margin">Los tipos de entidad soportados son:</p><ul><li><p class="no-margin">Facturas</p></li><li><p class="no-margin">Gastos</p></li><li><p class="no-margin">Contactos</p></li><li><p class="no-margin">Presupuestos</p></li><li><p class="no-margin">Albaranes</p></li><li><p class="no-margin">Transacciones</p></li><li><p class="no-margin">Productos</p></li></ul><p class="no-margin"></p>
<p class="no-margin">Para más información, consulta los artículos de webhooks:</p><ul><li><p class="no-margin"><b>Webhooks: recibe notificaciones en tiempo real</b> - visión general y catálogo de eventos</p></li><li><p class="no-margin"><b>Cómo configurar webhooks</b> - guía paso a paso para configurar endpoints desde el panel</p></li><li><p class="no-margin"><b>Verificar la firma de webhooks</b> - cómo verificar la autenticidad de las notificaciones</p></li><li><p class="no-margin"><b>API: Webhooks (endpoints y eventos)</b> - referencia de la API para gestionar webhooks por código</p></li></ul>
