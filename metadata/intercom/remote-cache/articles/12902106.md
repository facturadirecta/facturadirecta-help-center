<p class="no-margin">En FacturaDirecta, los <b>presupuestos, las facturas de venta y los albaranes</b> solo permiten asignar <b>un único método de pago</b> dentro del sistema. Esto significa que no es posible configurar dos cuentas bancarias diferentes mediante la opción estándar de método de pago.</p><p class="no-margin"></p>
<p class="no-margin">Si necesitas mostrar <b>dos o más números de cuenta</b> para que tu cliente pueda decidir dónde hacer la transferencia, puedes hacerlo añadiendo esta información como <b>texto informativo en el pie de la plantilla del documento</b>.</p><p class="no-margin"></p>
<h2 id="h_6e5011bd77">¿Cómo mostrar múltiples cuentas bancarias?</h2><ol><li><p class="no-margin">Entra en tu cuenta de FacturaDirecta.</p></li><li><p class="no-margin">Ve a <b>Configuración → Plantillas</b>.</p></li><li><p class="no-margin">Abre la plantilla que utilizas para presupuestos, facturas de venta o albaranes.</p></li><li><p class="no-margin">En el apartado <b>Textos &gt; Pie &gt; Texto personalizado</b>, añade el texto con las cuentas bancarias que quieras mostrar.</p></li></ol><p class="no-margin"><b>Ejemplo de texto para el pie:</b></p><pre><code>Puedes realizar el pago mediante transferencia a cualquiera de estas cuentas: - Cuenta principal: ES12 1234 5678 9012 3456 7890 - Cuenta alternativa: ES98 7654 3210 9876 5432 1098</code></pre><ol><li><p class="no-margin">Guarda los cambios.</p></li></ol><p class="no-margin">A partir de ese momento, cada vez que selecciones esa plantilla, el <b>PDF del documento</b> (presupuesto, factura o albarán) mostrará la información de todas las cuentas bancarias que hayas añadido, para que tu cliente pueda elegir cómo realizar el pago.</p><hr><h2 id="h_023a25d731">Recuerda</h2><p class="no-margin">✅ <b>Para mostrar UNA sola cuenta bancaria:</b> Usa el método de pago estándar explicado en <a href="https://help.facturadirecta.com/es/articles/1070840-mostrar-cuenta-bancaria-en-el-pdf-de-factura" target="_blank" class="intercom-content-link">este artículo</a></p><p class="no-margin">✅ <b>Para mostrar DOS O MÁS cuentas bancarias:</b> Usa el texto personalizado en el pie de la plantilla (como se explica en este artículo)</p><hr><p class="no-margin"></p>
