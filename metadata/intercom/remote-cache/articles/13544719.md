<p class="no-margin">El recurso <code>recurring/invoices</code> te permite crear automatizaciones que generan facturas de forma periódica. Es útil para servicios de suscripción, cuotas mensuales, mantenimientos anuales y cualquier facturación recurrente.</p><p class="no-margin"></p>
<p class="no-margin">Una factura recurrente es una automatización que contiene una plantilla de factura y un trigger (disparador) que define cuándo se ejecuta.</p><p class="no-margin"></p>
<p class="no-margin">💡 <b>Recomendación</b>: Antes de empezar a desarrollar, te recomendamos <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">crear un entorno de pruebas (sandbox)</a> para hacer tus primeras llamadas sin afectar tus datos reales.</p><p class="no-margin"></p>

<h1 id="h_estructura">Estructura de una factura recurrente </h1><p class="no-margin">Una factura recurrente tiene tres partes principales:</p><ol><li><p class="no-margin"><b>Configuración general</b>: título, estado activo/inactivo</p></li><li><p class="no-margin"><b>Trigger (disparador)</b>: cuándo se ejecuta la automatización</p></li><li><p class="no-margin"><b>Action (acción)</b>: qué factura se crea</p></li></ol><p class="no-margin"></p>
<h1 id="h_campos">Campos importantes </h1><h2 id="h_campos_main">Campos generales</h2><div class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Campo</b></p></td><td><p class="no-margin"><b>Tipo</b></p></td><td><p class="no-margin"><b>Descripción</b></p></td></tr><tr><td><p class="no-margin">title</p></td><td><p class="no-margin">string</p></td><td><p class="no-margin">Nombre descriptivo de la automatización</p></td></tr><tr><td><p class="no-margin">enabled</p></td><td><p class="no-margin">boolean</p></td><td><p class="no-margin">Si está activa (true) o pausada (false)</p></td></tr></tbody></table></div><h2 id="h_campos_trigger">Campos del trigger (rrule) </h2><p class="no-margin">El trigger usa el formato iCalendar RRULE para definir la recurrencia:</p><div class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Campo</b></p></td><td><p class="no-margin"><b>Tipo</b></p></td><td><p class="no-margin"><b>Descripción</b></p></td></tr><tr><td><p class="no-margin">freq</p></td><td><p class="no-margin">string</p></td><td><p class="no-margin">Frecuencia: yearly, monthly, weekly, daily</p></td></tr><tr><td><p class="no-margin">interval</p></td><td><p class="no-margin">number</p></td><td><p class="no-margin">Intervalo entre ejecuciones (1 = cada periodo, 2 = cada 2 periodos...)</p></td></tr><tr><td><p class="no-margin">bymonthday</p></td><td><p class="no-margin">number</p></td><td><p class="no-margin">Día del mes (1-31)</p></td></tr><tr><td><p class="no-margin">bymonth</p></td><td><p class="no-margin">number</p></td><td><p class="no-margin">Mes del año (1-12, solo para freq: yearly)</p></td></tr><tr><td><p class="no-margin">dtstart</p></td><td><p class="no-margin">string</p></td><td><p class="no-margin">Fecha de inicio (formato ISO 8601)</p></td></tr><tr><td><p class="no-margin">until</p></td><td><p class="no-margin">string</p></td><td><p class="no-margin">Fecha de fin (formato ISO 8601, opcional)</p></td></tr></tbody></table></div><h2 id="h_campos_action">Campos de la acción</h2><div class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Campo</b></p></td><td><p class="no-margin"><b>Tipo</b></p></td><td><p class="no-margin"><b>Descripción</b></p></td></tr><tr><td><p class="no-margin">action.type</p></td><td><p class="no-margin">string</p></td><td><p class="no-margin">Tipo de acción (siempre "createDocument")</p></td></tr><tr><td><p class="no-margin">action.template</p></td><td><p class="no-margin">object</p></td><td><p class="no-margin">Plantilla de la factura a crear (misma estructura que invoices)</p></td></tr></tbody></table></div><h1 id="h_frecuencias">Ejemplos de frecuencias</h1><div class="intercom-interblocks-table-container"><table role="presentation"><tbody><tr><td><p class="no-margin"><b>Frecuencia</b></p></td><td><p class="no-margin"><b>Configuración</b></p></td></tr><tr><td><p class="no-margin">Mensual (día 1)</p></td><td><p class="no-margin"><code>freq: "monthly", interval: 1, bymonthday: 1</code></p></td></tr><tr><td><p class="no-margin">Bimensual (día 15)</p></td><td><p class="no-margin"><code>freq: "monthly", interval: 2, bymonthday: 15</code></p></td></tr><tr><td><p class="no-margin">Trimestral (día 1)</p></td><td><p class="no-margin"><code>freq: "monthly", interval: 3, bymonthday: 1</code></p></td></tr><tr><td><p class="no-margin">Semestral (día 10)</p></td><td><p class="no-margin"><code>freq: "monthly", interval: 6, bymonthday: 10</code></p></td></tr><tr><td><p class="no-margin">Anual (10 enero)</p></td><td><p class="no-margin"><code>freq: "yearly", interval: 1, bymonthday: 10, bymonth: 1</code></p></td></tr><tr><td><p class="no-margin">Semanal (lunes)</p></td><td><p class="no-margin"><code>freq: "weekly", interval: 1, byday: "MO"</code></p></td></tr></tbody></table></div><h1 id="h_pausar">Pausar y reanudar</h1><p class="no-margin">Para pausar una factura recurrente, actualízala con <code>enabled: false</code>:</p><p class="no-margin"></p>
<pre><code>curl -X PUT \  -H "facturadirecta-api-key: TU_CLAVE_API" \  -H "Content-Type: application/json" \  -d '{    "content": {      "type": "automation",      "main": {        "enabled": false      }    }  }' \  https://app.facturadirecta.com/api/TU_COMPANY_ID/recurring/invoices/rec_xxxxx</code></pre><p class="no-margin"></p>
<p class="no-margin">Para reanudarla, usa <code>enabled: true</code>.</p><p class="no-margin"></p>
<h1 id="h_errores">Errores comunes </h1><h2 id="h_error_trigger">Error: "Invalid rrule configuration"</h2><p class="no-margin">La configuración del trigger no es válida. Verifica que:</p><ul><li><p class="no-margin"><code>freq</code> sea uno de: yearly, monthly, weekly, daily</p></li><li><p class="no-margin"><code>bymonthday</code> esté entre 1 y 31</p></li><li><p class="no-margin"><code>bymonth</code> esté entre 1 y 12 (si aplica)</p></li><li><p class="no-margin"><code>dtstart</code> sea una fecha válida en formato ISO 8601</p></li></ul><h2 id="h_error_template">Error: "Invalid template"</h2><p class="no-margin">La plantilla de factura no es válida. Verifica que tenga los campos obligatorios: contact, currency, lines, docNumber.</p><p class="no-margin"></p>
<h2 id="h_error_contact">Error: "contact not found"</h2><p class="no-margin">El cliente indicado en la plantilla no existe. Verifica el ID del contacto.</p><p class="no-margin"></p>
