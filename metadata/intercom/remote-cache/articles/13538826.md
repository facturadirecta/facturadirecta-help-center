<p class="no-margin">Esta guía te ayuda a hacer tu primera llamada a la API de FacturaDirecta. En pocos pasos tendrás una conexión funcional.</p><p class="no-margin"></p>
<p class="no-margin">💡 <b>Recomendación</b>: Haz tus primeras pruebas en un <a href="https://help.facturadirecta.com/es/articles/12608263-donde-puedo-crear-un-entorno-de-pruebas-sandbox-para-la-api" target="_blank" class="intercom-content-link">entorno de pruebas (sandbox)</a> antes de usar la API en producción. Los sandboxes tienen datos aislados y puedes experimentar sin afectar tu empresa real.</p><p class="no-margin"></p>
<h1 id="h_requisitos">Requisitos previos</h1><p class="no-margin">Antes de empezar, necesitas:</p><ul><li><p class="no-margin">Una cuenta en <a href="https://app.facturadirecta.com" target="_blank" class="intercom-content-link">FacturaDirecta</a> (el plan gratuito es suficiente)</p></li><li><p class="no-margin">Tu <b>companyId</b> (identificador de empresa)</p></li><li><p class="no-margin">Una <b>clave de API</b> con los permisos necesarios </p></li></ul><p class="no-margin">👉 Si aún no tienes una clave de API, consulta <a href="https://help.facturadirecta.com/es/articles/13538824-como-crear-claves-de-api" target="_blank" class="intercom-content-link">Cómo crear claves de API</a>.</p><p class="no-margin"></p>
<h1 id="h_primera_llamada">Tu primera llamada</h1><p class="no-margin">Vamos a listar tus facturas. Abre una terminal y ejecuta:</p><p class="no-margin"></p>
<pre><code>curl -H "facturadirecta-api-key: TU_CLAVE_API" \     https://app.facturadirecta.com/api/TU_COMPANY_ID/invoices</code></pre><p class="no-margin"></p>
<p class="no-margin">Sustituye:</p><ul><li><p class="no-margin"><code>TU_CLAVE_API</code> por tu clave de API</p></li><li><p class="no-margin"><code>TU_COMPANY_ID</code> por tu identificador de empresa</p></li></ul><p class="no-margin"></p>
<h1 id="h_entender_respuesta">Entender la respuesta</h1><p class="no-margin">Si todo va bien, recibirás una respuesta JSON con tus facturas:</p><p class="no-margin"></p>
<pre><code>{  "data": [    {      "id": "inv_abc123",      "number": "2026-001",      "date": "2026-01-15",      "contact": {        "id": "con_xyz789",        "name": "Cliente Ejemplo S.L."      },      "total": 121.00,      "currency": "EUR",      "state": "sent"    }  ],  "pagination": {    "page": 1,    "perPage": 25,    "total": 150  }}</code></pre><p class="no-margin"></p>
<p class="no-margin">La respuesta incluye:</p><ul><li><p class="no-margin"><b>data</b>: Array con los registros</p></li><li><p class="no-margin"><b>pagination</b>: Información para paginar resultados</p></li></ul><h1 id="h_crear_factura">Siguiente paso: crear una factura</h1><p class="no-margin">Una vez que puedas leer datos, el siguiente paso es crear registros. Este ejemplo crea una factura básica:</p><p class="no-margin"></p>
<pre><code>curl -X POST \     -H "facturadirecta-api-key: TU_CLAVE_API" \     -H "Content-Type: application/json" \     -d '{       "contactId": "con_xyz789",       "date": "2026-01-27",       "lines": [         {           "description": "Servicio de consultoría",           "quantity": 1,           "unitPrice": 100.00         }       ]     }' \     https://app.facturadirecta.com/api/TU_COMPANY_ID/invoices</code></pre><p class="no-margin"></p>
<h1 id="h_errores_comunes">Errores comunes </h1><h2 id="h_error_401">Error 401: Unauthorized</h2><p class="no-margin">La clave de API no es válida o no está presente. Verifica:</p><ul><li><p class="no-margin">Que la cabecera se llama <code>facturadirecta-api-key</code> (no <code>Authorization</code>)</p></li><li><p class="no-margin">Que la clave está copiada correctamente</p></li><li><p class="no-margin">Que la clave no ha sido eliminada</p></li></ul><p class="no-margin"></p>
<h2 id="h_error_403">Error 403: Forbidden</h2><p class="no-margin">La clave no tiene permisos para esta operación. Verifica:</p><ul><li><p class="no-margin">Que la clave tiene el permiso correspondiente (ej: <code>invoices:read</code> para leer facturas)</p></li><li><p class="no-margin">Que el companyId es de la empresa donde se creó la clave</p></li></ul><p class="no-margin"></p>
<h2 id="h_error_404">Error 404: Not Found</h2><p class="no-margin">El recurso no existe. Verifica:</p><ul><li><p class="no-margin">Que el companyId es correcto</p></li><li><p class="no-margin">Que la ruta del endpoint es correcta</p></li><li><p class="no-margin">Que el ID del recurso existe</p></li></ul><h1 id="h_siguientes_pasos">Siguientes pasos</h1><p class="no-margin">Ahora que tienes una conexión funcionando:</p><ul><li><p class="no-margin">Explora la <a href="https://www.facturadirecta.com/api/" target="_blank" class="intercom-content-link">especificación OpenAPI</a> para ver todos los endpoints disponibles</p></li><li><p class="no-margin">Importa la especificación en <b>Postman</b> o <b>Insomnia</b> para probar los endpoints de forma visual</p></li><li><p class="no-margin">Genera un cliente para tu lenguaje de programación con herramientas como <b>OpenAPI Generator</b></p></li></ul><h1 id="h_articulos_relacionados"></h1>
